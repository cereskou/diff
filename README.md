# diff
compare two directories using sqlite memory database.

```
Usage:
  diff.exe [OPTIONS]

Application Options:
  /s, /source:   source directory
  /t, /target:   target directory
  /o, /output:   output file
  /v, /version   show version

Help Options:
  /?             Show this help message
  /h, /help      Show this help message
```
